# Docs Graph Context

The docs graph report lives at `.kody/reports/docs-graph.md` on the `kody-state` branch.

Use it to understand documentation topology:

- markdown files in the repo
- local markdown links and external domains
- broken local links
- docs without an H1
- docs with TODO/FIXME markers

The graph is a state report. It should be read before documentation cleanup, onboarding work, or changes that move files referenced by docs.
